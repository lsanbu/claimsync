Daily cycle

What happens
every morning

01
06:00 UAE
Azure cron fires
Container App Job starts. Active facilities fetched from PostgreSQL.
02
06:00 – 06:08
Shafafiya sync per facility
Day split into 12 × 2-hour intervals. SearchTransactions → DownloadTransactionFile for each.
03
06:08
Classify & store
Files routed to claims/, remittance/, resubmission/, logs/. Full audit trail in PostgreSQL.
04
07:00 UAE
Client pull via secure exe
ClaimSyncPull.exe fetches files to local Reggr\ folder. JWT auth only — no API keys on client machine.
